Capputtino Shop

-Este é o site da cafeteria Capputtino Shop onde o cliente pode acessar para saber informações da loja como endereço e horários, ver algumas das receitas que a cafeteria usa e fazer reservas de mesas.

Tecnologias Utilizadas:
-HTML5
-CSS3
-Google Fonts

Planos de melhoria do projeto:
-Um cardápio digital com imagens dos produtos e descrições sobre quais igredientes são usados.
-Tema escuro para o site.
-Botão que leva o cliente para o whatsapp da loja.
-Bordar personalizadas para os iframes.
-Área para colocar comentários e avaliações dos clientes.
-Adicionar localização da cafeteria com google maps.
-Área de login para guardar emails dos clientes para o remarketing com promoções e eventos ou um form para reter apenas os emails e dar um cupom para o cliente com o mesmo intuito de remarketing.
